## About Me

**Computer Engineering** student at **Covenant University**, with a strong interest in the space where media and technology meet. I enjoy turning ideas into clean, interactive interfaces and visual solutions that are both functional and engaging.

### What Drives Me

I'm driven by solving problems, whether that means writing code for a responsive interface or putting together a visual that communicates clearly. I like work that blends logic with creativity, where frontend development and design come together to build things people genuinely enjoy using.
